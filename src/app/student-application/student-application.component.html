Student Application

Thanks for your interest in UW Blueprint! We are always looking for UW students to take part in our club activities and join our talented team.

We are looking for the following positions:

Project developer

A project developer works on a specific project for a non-profit, led by a team lead. This includes mostly dev work, and potentially some architecture design, and/or visual design.

Project developer (mentor)

A project developer who is also a mentor has the same duties as a project developer, but in addition, they pair program with an inexperienced developer.

Project lead

A project lead is responsible for organizing other team members and getting the project done. This can include creating tasks, assigning priorities and giving time estimates, or just following up with team members to make sure work is on track.
